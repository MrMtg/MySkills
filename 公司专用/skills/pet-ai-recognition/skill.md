## 开发流程

当用户要求开发新的识别页面时，按以下步骤进行：

1. **确认配置**: SUB_MODULE_TYPE、PET_PROFILE_ID
2. **复制徽章系统**: HTML + CSS + JavaScript
3. **接入识别流程**: session-start → upload-image → generate-report
4. **报告页处理**: 删除不需要的元素（品种、性别等，用户会说明）
5. **测试验证**: 徽章显示、识别流程
